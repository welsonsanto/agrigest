# 🌾 AgriGest - Sistema de Gestão do Agronegócio

Sistema completo de gestão para fazendas e agronegócio, desenvolvido em React.

## Funcionalidades

- Dashboard com visão geral
- Cadastro de Fazenda, Clientes, Transportadoras, Fornecedores
- Gestão de Grãos, Talhões e Produtividade
- Romaneios de Entrada e Saída
- Controle de Insumos e Estoque
- Contratos e Agendamentos
- Relatórios de Motoristas e Colheita
- Sistema de Login com controle de acesso
